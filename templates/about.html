About Cloudwich

This site is running Cloudwich. A simple Django based wrapper around various web services, namely Picasa, Tumblr, Facebook and Disqus.

The idea was inspired by the PHP project Pubwich, which does a very similar thing using PHP. Cloudwich wraps things a bit deeper than Pubwich. But mostly I just wanted to play with Django, so that's why I built Cloudwich.

The default theme was inspired / stolen (which is it these days) from the fantastic Tumblr theme Cavalcade by Peter Vidani.

You can find the source code and build your own Cloudwich site in no time flat, by grabbing the latest version from github.
